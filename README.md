# 🧠 Kiosk Backend

AI 기반 어르신 음성인식 키오스크 프로젝트의 백엔드 서버입니다.

---

## ⚙️ 기술 스택

- Java 17
- Spring Boot 3.x
- Spring Web / JPA
- H2 Database (개발용)

---

## 🔧 주요 기능

- 음성 명령 처리 API  
- 키오스크 시나리오 관리  
- TTS/NLP 결과 연결 인터페이스
